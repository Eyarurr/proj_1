Oculus Go - автономная VR гарнитура и требует определенных действий для установки приложения Biganto. Чтобы установить Biganto Tour Player, вам понадобится ключ пользователя, который можно сгенерировать на этой странице и Oculus Go, сопряженный с мобильным устройством.

Важно: Oculus Go больше не поддерживается Oculus. Данная версия приложения окончательная и не будет изменяться в дальнейшем. Приложение для Go перестанет быть доступно в какой то момент в будущем.

Сопряжение Oculus Go с мобильным устройством

Если ваши устройства уже сопряжены, вы можете перейти к генерации ключей и установке приложения, в противном случае, следуйте данным инструкциям:

  1. Установите приложение Oculus на мобильное устройство и создайте аккаунт для получения доступа в Oculus store
  2. Откройте приложение и выберите “Settings” в нижней части экрана
  3. Выберите “Pair New Headset”

  4. Выберите Oculus Go из списка и следуйте инструкциям приложения

Установка приложения

Когда устройство будет готово, вы можете перейти к установке приложения:

  1. Чтобы установить приложение, вам понадобится ключ пользователя. Мы рекомендуем генерировать ключ сразу в браузере мобильного устройства, чтобы упростить копирование в мобильное приложение Oculus. Нажмите “Сгенерировать ключ” ниже:

  2. Откройте приложение Oculus и перейдите в “Settings”
  3. Пролистайте страницу ниже и выберите “Redeem Code”

  4. Вставьте сгенерированный ключ пользователя в поле и нажмите “Submit”

  5. Вы увидите следующий экран при успешной активации по коду:

  6. Нажмите на Biganto Tour Player и выберите “Install on Oculus Go” в нижней части экрана. Загрузка начнется в ближайшее время.

Доступ к приложению из меню VR гарнитуры

Чтобы открыть приложение в меню Oculus Go, выполните следующее:

  1. Включите Oculus Go, нажмите кнопку “Oculus” на контроллере.
  2. Выберите с помощью контроллера пункт “Navigate” в меню, выберите “Library” чтобы открыть список доступных приложений

  3. Выберите “Biganto Tour Player” из списка. Войдите в аккаунт Biganto чтобы получить доступ к вашим турам.

Oculus Quest - автономная VR гарнитура и требует определенных действий для установки приложения Biganto. Чтобы установить Biganto Tour Player на ваш Oculus Quest достаточно открыть страницу приложения в OculusStore. Откройте страницу через Oculus Quest или связанного мобильного устройства и нажмите "Get" для установки приложения.

Альтернативная установка через SideQuest

Для установки Biganto Tour Player вам потребуется:

  1. Установить альтернативный загрузчик приложений SideQuest на PC
  2. Кабель для передачи данных с PC на Oculus Quest (обратите внимание, что кабель поставляемый в комплекте с Quest не подходит для этих целей)
  3. Смартфон с установленным приложением Oculus

SideQuest это альтернативное средство загрузки приложений и требует наличия PC. SideQuest устанавливается на PC и дает возможность устанавливать приложения по USB кабелю на Oculus Quest. К сожалению установка приложений без кабеля, непосредственно на VR гарнитуру невозможна по причине конфликтов с пользовательским соглашением Quest и по соображениям безопасности.

После установки SideQuest и выполнения всех необходимых инструкций вы можете скачать приложение Biganto из маркета приложений SideQuest.

Важно: SideQuest не является официальным продуктом Oculus, его установка требует определенных знаний PC. Если вы не уверены в том, что вы делаете, мы не рекомендуем выполнять установку.
