Краткое руководство

  1. Перетяните скрипт Virtoaster camera tool во вьюпорт сцены для его запуска
  2. Нажмите “Point”, поместите вспомогательные объекты туда, где вы хотите перемещаться по туру. Сохраняйте дистанцию 1,5-3м между ними.
  3. Поместите все вспомогательные объекты на высоту 1,5 м над полом
  4. Нажмите “Create camera”
  5. Нажмите “Select camera” и выберите созданную камеру, затем нажмите “Apply Virtoaster render settings”
  6. Перейдите к разделу “Настройка рендера”

Полное руководство

Для создания Virtoaster тура необходимы два основных компонента: набор панорамных 360 рендеров и 3d модель сцены. Здесь мы расскажем как правильно выполнить рендер и начнем с настройки камеры.

Чтобы установить камеру, загрузите наш скрипт для её настройки. Откройте сцену и перейдите во вьюпорт в проекции Top (горячая клавиша T). Перетащите скрипт камеры во вьюпорт, вы увидите следующий интерфейс:

Начните с создания вспомогательных объектов типа “Point”. Нажмите “Point” в интерфейсе скрипта и используйте левую клавишу мыши чтобы расставить объекты в те места, в которые вы хотите перемещаться в туре. Нажмите правую клавишу мыши чтобы завершить.

Каждый объект “Point” соответствует одному рендеру, поэтому расставляйте точки только там, где вы хотите перемещаться. Расставляйте точки только в тех местах, где физически возможно перемещаться, например не ставьте точки над столами, шкафами и т.д.

Важно: не расставляйте точки слишком близко друг к другу, т.к. тогда будет не удобно перемещаться по туру. Мы советуем соблюдать дистанцию между точками в 1,5-3 метра.

Перемещение между точками будет возможно, только если между ними есть линия прямой видимости. Между такими точками не должно быть препятствий, таких как закрытые двери, шторы, экраны и т.д.

Вкратце, основные требования к расстановке точек:

  1. Расстояние между соседними точками должно быть не более трёх метров.
  2. Точки, между которыми будет переход, должны находиться в прямой видимости.
  3. Точки должны быть расставлены только в местах, где возможно перемещение (например, не должно быть точек над столами, постелью и т.п.).
  4. Точки должны быть распределены равномерно по помещению.

Пример правильного расположения точек:

Далее, поместите все созданные точки на высоту 1,5 метра над полом (средняя высота, с которой смотрит человек). При наличии лестниц и нескольких этажей в вашей сцене не забудьте выставить высоту соответственно.

Наконец, выберите все точки и нажмите “Create camera”. Будет создана анимированная камера, каждая точка будет соответствовать одному кадру.

Важно: по умолчанию 3d Max создает объекты “Point” на нулевой высоте. Убедитесь, что высота точек выставлена правильно перед тем как создавать камеру.

В дополнение к камере для каждой точке будет создана дополнительная геометрия. Она используется для определения координат кадра. Данная геометрия не влияет на рендер.

Важно: если по какой либо причине вам необходимо изменить положение точки, сперва удалите камеру и дополнительную геометрию. Затем измените позицию точки и снова нажмите “Create camera”.

Важно: если в вашей сцене есть анимированные объекты, скрипт предложит удалить анимацию из сцены:

Анимация может привести к разному положению объектов на различных кадрах, например к открытой\закрытой двери на соседних кадрах тура и, соответственно несовпадению с 3d моделью. Таким образом в сцене может быть единственный анимированный объект - камера, созданная с помощью наших инструментов.

Нажмите “Select camera” и выберите созданную камеру, затем нажмите “Apply Virtoaster render settings”. Скрипт автоматически установить необходимые настройки рендера и выведет сообщение о внесенных изменениях:

Скрипт установил основные настройки рендера и, хотя вы уже можете начать рендер, мы настоятельно советуем изучить раздел “Настройка рендера”

Опциональный этап:

По умолчанию координаты точек определяются с помощью дополнительной геометрии, создаваемой с помощью наших инструментов. Вы также можете загрузить координаты точек вручную, о чем подробнее рассказано в статье “Загрузка контента”.

Чтобы выгрузить координаты точек в .txt файл, воспользуйтесь скриптом Virtoaster Cam tool. Нажмите “Select camera” и выберите Virtoaster-камеру, затем нажмите “Create”. Вам будет предложено выбрать где сохранить файл с координатами на вашем компьютере.
